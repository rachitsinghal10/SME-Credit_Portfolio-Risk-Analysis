Project Title
SME Credit Portfolio Risk Analysis Dashboard

Objective
Analyze the risk profile of an SME loan portfolio using Excel by evaluating exposure distribution, asset quality, sector concentration, and borrower concentration risk.

Dataset Description
The dataset contains simulated SME loan portfolio data with the following fields:
* Account ID
* Branch
* Sector
* Outstanding Amount
* Risk Category
* NPA Flag

Key Metrics Calculated
* Total Portfolio Exposure
* Gross NPA %
* High Risk Exposure %
* Branch-wise NPA %
* Sector Exposure Distribution
* Top 5 Borrower Exposure %
* Top 10 Borrower Exposure %

Key Insights
Example insights:
* Gross NPA ratio of ~3% indicates stable portfolio quality.
* High-risk exposure remains limited at ~6.6%.
* Borrower concentration is low, with Top 10 borrowers accounting for ~6% of total exposure.
* Delhi branch contributes the highest portfolio exposure.
* Kolkata branch exhibits the highest NPA ratio.

Tools Used
* Microsoft Excel
* SUMIFS
* INDEX + MATCH
* Data visualization using Excel charts
